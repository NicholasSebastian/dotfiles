# My Dotfiles

These are the dotfiles used to create my current workspace setup on my Asus Zephyrus G14.
It represents a Windows 11 environment running WSL2 with Ubuntu 24.04.

With common sense, you'd know that the .ps1 file is to be executed on Powershell from Windows,
whereas the .sh file is to be executed on WSL2 from Ubuntu.

Likewise, the folders for AppData and ProgramFiles fall under the Windows filesystem,
whereas the .config folder falls under the Linux filesystem.
